
## 📁 Projects Overview

Below is a list of all projects included in this repository.  
Each project has its own folder, documentation, and Docker setup.

### **Project 01 — Chatbot** *(In Progress)*  
A simple but extensible AI‑powered chatbot built with Python and Gradio.  
This project demonstrates:

- a clean conversational interface  
- integration with an LLM provider (OpenAI, LM Studio, Ollama, etc.)  
- context‑aware responses  
- a modular architecture with a dedicated AI client  
- full Docker support for easy deployment
